# 奇门遁甲排盘工具

## 概述

本技能提供完整的时家奇门遁甲排盘功能，支持任意时间的奇门局起盘和分析。

## 核心功能

1. **时间转换**：公历转农历、干支纪年月日时
2. **节气计算**：精确计算24节气时刻
3. **起局判断**：阴遁/阳遁、超神/接气、局数计算
4. **九宫排盘**：天盘、地盘、人盘、神盘完整布局
5. **格局分析**：吉凶判断、用神分析

## 使用方法

### 基本排盘

当用户请求奇门遁甲排盘时，执行以下步骤：

1. 获取排盘时间（默认当前时间，或用户指定时间）
2. 运行排盘脚本：
   ```bash
   python3 scripts/qimen_paipan.py "YYYY-MM-DD HH:MM"
   ```
3. 展示排盘结果，包括：
   - 基本信息（时间、干支、节气、局数）
   - 九宫格局图
   - 各宫详细信息
   - 格局分析建议

## 分析要点

排盘后，根据用户需求提供分析：

- **预测占卜**：看用神宫位、门星组合、格局吉凶
- **择吉选时**：找吉门吉星吉神组合的时辰
- **方位选择**：根据吉宫确定有利方位

## 注意事项

- 排盘以真太阳时为准，需考虑地理经度时差
- 节气交接时刻前后需特别注意局数变化
- 不同流派在超神接气、拆补等细节上有差异，本工具采用主流时家奇门规则
